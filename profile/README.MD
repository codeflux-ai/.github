## About Us

Codeflux AI is a solo-dev product studio that builds **real, deployed software** using AI-assisted vibe coding. We don't build demos or proof-of-concepts — every project in this organisation is **live, maintained, and solving actual problems** for real users.

Our philosophy is simple: **zero-cost architecture, maximum impact.** Every project runs on free-tier infrastructure. The only investment is time, creativity, and imagination.

## Our Projects

### 1. VRK Diet Companion

> Personalised wellness tracking built on the VRK Diet system

| | |
|---|---|
| **Live App** | [vrk-diet-companion.vercel.app](https://vrk-diet-companion.vercel.app/) |
| **Stack** | React.js, PWA, Vercel |
| **Status** | Live &middot; Actively maintained &middot; Native app in progress |

**What it does:** A comprehensive diet companion app that helps users track meals, monitor weight trends, and log daily hydration — all aligned to the phased structure of the VRK Diet. Users can follow their diet journey with structured phase guidance and real-time progress insights.

**Why we built it:** The VRK Diet has a dedicated following but lacked a purpose-built digital companion. Generic calorie trackers don't understand phase-based dieting. This app fills that gap with a focused, distraction-free experience designed for the VRK community.

**What's next:** Currently being converted from a Progressive Web App into a fully native mobile application for a smoother, more performant experience on iOS and Android.

### 2. AquaRemind

> Smart hydration and supplement reminders

| | |
|---|---|
| **Live App** | [aqua-hydrate.vercel.app](https://aqua-hydrate.vercel.app/) |
| **Stack** | React.js, PWA, Vercel |
| **Status** | Live &middot; Actively maintained |

**What it does:** A lightweight wellness app that sends configurable reminders for water intake and supplement schedules. Designed for quick daily interactions — open, log, done. No bloat, no unnecessary features, just the essentials to keep you hydrated and on track.

**Why we built it:** Staying hydrated sounds simple, but most people forget. Existing reminder apps are either too complex or buried under unrelated features. AquaRemind is a single-purpose tool that does one thing well — it nudges you to drink water and take your supplements on time.

### 3. AI Security Auditor

> AI-powered code security analysis

| | |
|---|---|
| **Live App** | [ai-security-auditor.lovable.app](https://ai-security-auditor.lovable.app/ ) | 
| **Stack** | React.js, Google Gemini API, Supabase, Vercel |
| **Status** | Built &middot; Deployed |

**What it does:** A web application that leverages Google Gemini's API to perform intelligent security audits and vulnerability assessments on codebases. Paste your code, get actionable security insights powered by AI — complete with severity ratings and remediation suggestions.

**Why we built it:** Security auditing is often expensive, time-consuming, or requires deep specialist knowledge. This tool democratises code security by making AI-powered vulnerability scanning accessible to solo developers and small teams. Built entirely on free-tier infrastructure with quota guardrails and usage warnings baked in to keep it sustainable.

## How We Build

We follow a streamlined AI-assisted development pipeline that takes ideas from concept to production rapidly:

| Step | Tool | Purpose |
|:-----|:-----|:--------|
| **Scaffold** | Lovable | Rapid UI/UX prototyping and initial structure |
| **Refine** | Cursor | AI-assisted code refinement and iteration |
| **Iterate** | Claude Code | Complex logic, debugging, and architecture |
| **Deploy** | Vercel | CI/CD with seamless GitHub integration |
| **Maintain** | — | Regular updates driven by real user feedback |

This pipeline allows us to go from idea to deployed product in hours, not weeks — without sacrificing code quality or user experience.

## Principles

- **Ship, don't polish** — A deployed MVP beats a perfect prototype sitting in localhost

- **Real users, real problems** — Every project solves a genuine need, not a hypothetical one
- **Solo doesn't mean small** — One developer with the right tools can build what used to take a team
